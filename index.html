Course Summary

This course provides a grounding in computational thinking, assuming no prior background in programming. It covers Python and uses real-world examples to illustrate concepts.

Student Learning Outcomes

  • Use a spreadsheet for simple data management tasks.
  • Write Python programs to solve simple computational problems.
  • Solve problems via computer systems and build simple human-centered interfaces.
